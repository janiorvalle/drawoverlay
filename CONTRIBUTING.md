# Contributing

Install Node.js 22 or newer and pnpm 11, then run:

```sh
pnpm install
pnpm verify
```

All changes require tests and a clean verification run. Keep runtime code free
of network calls and dependencies. Do not commit secrets, environment files,
private URLs, personal filesystem paths, or planning documents.

By contributing, you agree that your contribution is licensed under the MIT
License and the project's contributor license agreement.
